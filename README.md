## ðŸ˜Š How to run

Before starting, you need to have [Git](https://git-scm.com) and [Node](https://nodejs.org/en/) installed on your machine.

```sh
- Clone this repo:
$ git clone https://github.com/marcelopajr/ignite-timer.git

- Enter directory:
$ cd ignite-timer

- Install dependencies:
$ npm install

- Launch the app:
$ npm run dev
```
